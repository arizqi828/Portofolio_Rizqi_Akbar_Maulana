🦑 Squid Stock Assessment — WPP-NRI 573
Independent Project · R Studio · ggplot2 · tidyverse · minpack.lm
Penilaian status stok cumi-cumi di WPP-573 (Samudera Hindia Selatan Jawa) menggunakan data logbook KKP 2018–2024.

📉 Biomassa turun drastis dari 2.000 ton (2018) ke hampir nol di 2024–2025 — jauh di bawah BMSY (1.162 ton)
⚠️ 5 dari 7 tahun tangkapan aktual melampaui MSY (549 ton) — indikasi overfishing
